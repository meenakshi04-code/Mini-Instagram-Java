# Mini Instagram (Java Console App)

This is a simple console-based Mini Instagram application built using Java. It allows users to create accounts, post content, and like posts.

## Features
- Create user accounts
- Create posts
- View all posts
- Like posts
- Menu-driven interface

## Technologies Used
- Java
- OOP (Classes & Objects)
- ArrayList (for posts)
- HashSet (for users)
- Scanner (user input)

## How to Run

1. Compile:
   javac MiniInstagram.java

2. Run:
   java MiniInstagram

## Example

- Create a user
- Add posts
- View posts
- Like posts
